## Common Patterns Across All Papers

### Table of Contents
✅ 7 out of 8 papers have a dedicated TOC page (usually page 2)
- Helps judges navigate quickly
- Shows professional organization

### Summary Sheet
✅ All papers have 1-page summary sheet (page 1)
- Required by MCM rules
- Not counted in page limit

### Section Numbering
✅ Most papers use hierarchical numbering (1, 1.1, 1.1.1)
- Facilitates cross-referencing
- Professional appearance

### Figure Quality
✅ All O-Prize papers have high-quality figures
- Clear axis labels
- Descriptive captions
- Consistent color schemes
- Multiple panels where appropriate

### Code in Appendix
✅ 6 out of 8 papers include code in appendix
- Usually Python or MATLAB
- Well-commented
- Key algorithms only (not all code)

### Sensitivity Analysis
✅ ALL papers include explicit sensitivity analysis
- Usually 2-4 pages
- Parameter sweep graphs
- Robustness discussion

### Strengths/Weaknesses
✅ ALL papers include honest self-assessment
- Usually 1-2 pages
- Critical but constructive
- Shows maturity and scholarship
