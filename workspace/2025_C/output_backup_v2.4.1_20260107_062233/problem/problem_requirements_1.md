# MCM 2025 Problem C: 需求分析 v1

## 问题标题

**Models for Olympic Medal Tables**

## 问题概述

这是一个关于建立奥运会奖牌榜预测模型的问题。核心背景是：
- 奥运会奖牌榜是各国关注的焦点
- 需要建立数学模型来预测和解释各国的奖牌数量
- 必须基于提供的历史数据（1896-2024年夏季奥运会）
- 需要预测2028年洛杉矶奥运会奖牌榜

问题的目标包括：
1. 建立奖牌计数预测模型
2. 预测2028年奥运会奖牌分布
3. 分析"优秀教练"效应
4. 发现其他原创洞察

---

## 主要需求

### 需求 1: [ ] 开发奖牌计数模型
- 至少包括金牌数和总奖牌数
- 必须包含模型预测的不确定性/精度估计
- 必须包含模型性能评估指标

### 需求 2: [ ] 预测2028年洛杉矶奥运会奖牌榜
- 提供所有结果的预测区间（prediction intervals）
- 识别哪些国家可能进步
- 识别哪些国家可能表现不如2024年

### 需求 3: [ ] 首次获奖国家的预测
- 模型必须包含尚未获得过奖牌的国家
- 预测有多少国家将在下届奥运会获得首枚奖牌
- 提供该估计的赔率/概率

### 需求 4: [ ] 分析奥运会项目与奖牌数的关系
- 探索项目数量和类型与各国奖牌收入的关系
- 识别对各国最重要的运动项目
- 解释为什么这些运动重要
- 分析主办国选择的项目如何影响结果

### 需求 5: [ ] 研究"优秀教练"效应
- 使用提供的数据寻找教练效应的证据
- 估计教练效应对奖牌数的贡献程度
- 选择3个国家并识别他们应该投资"优秀教练"的运动项目
- 估计这种投资的潜在影响

### 需求 6: [ ] 提供其他原创洞察
- 模型揭示的其他关于奥运会奖牌数的洞察
- 解释这些洞察如何告知各国奥委会

## 初步观察

### 问题特征
1. **时间序列预测问题**: 需要基于历史数据预测未来（2028年）
2. **多层次建模**: 需要在国家、运动项目、时间等多个层次建模
3. **不确定性量化**: 强调预测区间和不确定性估计
4. **因果推断**: 教练效应分析涉及因果推断
5. **面板数据**: 同时包含多个国家和多个时间点的数据

### 数据特点
1. **不平衡数据**: 各国参赛历史长度不同，数据量差异大
2. **稀疏性**: 许多国家在某些年份没有参赛或没有奖牌
3. **零膨胀**: 大量国家奖牌数为0
4. **时间趋势**: 随着时间推移，参赛国家和项目数量都在增加

### 建模挑战
1. **零值处理**: 如何处理尚未获奖或奖牌为0的国家
2. **因果关系**: 如何区分相关性和因果性（特别是教练效应）
3. **外生变量**: 如何纳入主办国、项目数量等外生因素
4. **模型选择**: 需要平衡预测精度和可解释性
5. **验证策略**: 如何验证2028年的预测（没有真实标签）

### 方法论考虑
1. 可能需要使用面板数据模型（Panel Data Models）
2. 考虑贝叶斯方法以获得预测区间
3. 可能需要使用零膨胀模型（Zero-inflated models）
4. 时间序列方法（ARIMA, 状态空间模型等）
5. 机器学习方法（随机森林、梯度提升等）用于预测
